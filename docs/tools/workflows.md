# Workflows Tools (5)

Enable with `--caps=workflows`.

| Tool | Description |
| --- | --- |
| `wavexis_multi_action` | Execute a YAML sequence of actions in one call |
| `wavexis_raw_cdp` | Send raw CDP command (escape hatch) |
| `wavexis_raw_bidi` | Send raw WebDriver BiDi command (escape hatch) |
| `wavexis_browser_context_create` | Create an isolated browser context |
| `wavexis_browser_context_close` | Close a browser context |
