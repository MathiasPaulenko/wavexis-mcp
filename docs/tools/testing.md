# Testing Tools (4)

Enable with `--caps=testing`.

| Tool | Description |
| --- | --- |
| `wavexis_assert_visible` | Assert element is visible |
| `wavexis_assert_text_visible` | Assert text is visible on page |
| `wavexis_assert_url` | Assert current URL matches expected |
| `wavexis_generate_locator` | Generate robust CSS selector for an element |
