# Video Tools (4)

Enable with `--caps=video`.

| Tool | Description |
| --- | --- |
| `wavexis_video_record` | Start video recording |
| `wavexis_video_stop` | Stop recording and get video |
| `wavexis_video_add_chapter` | Add chapter marker |
| `wavexis_video_action_overlay` | Add action overlay on video |
