# Interactions Tools (5)

Enable with `--caps=interactions`.

| Tool | Description |
| --- | --- |
| `wavexis_dialog_accept` | Accept a JavaScript dialog (alert, confirm, prompt) |
| `wavexis_dialog_dismiss` | Dismiss a JavaScript dialog |
| `wavexis_intercept_download` | Intercept a file download |
| `wavexis_grant_permission` | Grant browser permission (geolocation, notifications, etc.) |
| `wavexis_reset_permissions` | Reset all granted permissions |
