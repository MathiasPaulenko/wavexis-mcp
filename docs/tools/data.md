# Data Tools (6)

Enable with `--caps=data`.

| Tool | Description |
| --- | --- |
| `wavexis_record` | Record browser actions to YAML (codegen) |
| `wavexis_lighthouse` | Run Lighthouse audit (performance, accessibility, SEO, best-practices) |
| `wavexis_extract` | Extract structured data from a page using CSS selectors |
| `wavexis_websocket_intercept` | Intercept WebSocket messages |
| `wavexis_crawl` | Crawl multiple URLs and extract content |
| `wavexis_visual_diff` | Compare two screenshots for visual regression |
