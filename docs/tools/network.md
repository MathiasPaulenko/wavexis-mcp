# Network Tools (9)

Enable with `--caps=network`.

| Tool | Description |
| --- | --- |
| `wavexis_set_headers` | Set HTTP headers for all requests |
| `wavexis_set_user_agent` | Set custom User-Agent |
| `wavexis_block_requests` | Block requests by URL pattern |
| `wavexis_throttle_network` | Throttle network (preset or custom) |
| `wavexis_set_cache_disabled` | Disable browser cache |
| `wavexis_capture_har` | Capture HAR file |
| `wavexis_intercept_requests` | Intercept and modify requests |
| `wavexis_mock_response` | Mock a response for a URL pattern |
| `wavexis_network_requests` | List captured network requests |
