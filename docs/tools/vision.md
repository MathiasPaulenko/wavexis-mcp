# Vision Tools (6)

Enable with `--caps=vision`.

Coordinate-based mouse operations for pixel-precise interaction.

| Tool | Description |
| --- | --- |
| `wavexis_mouse_move` | Move mouse to coordinates |
| `wavexis_mouse_down` | Press mouse button at coordinates |
| `wavexis_mouse_up` | Release mouse button at coordinates |
| `wavexis_mouse_click_xy` | Click at specific x,y coordinates |
| `wavexis_mouse_double_click_xy` | Double-click at coordinates |
