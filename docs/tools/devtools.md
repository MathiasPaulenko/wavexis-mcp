# DevTools Tools (23)

Enable with `--caps=devtools`.

## Performance

| Tool | Description |
| --- | --- |
| `wavexis_perf_metrics` | Get performance metrics (LCP, FCP, CLS, TTFB) |
| `wavexis_perf_trace` | Capture a performance trace |
| `wavexis_perf_profile` | Capture a CPU profile |
| `wavexis_perf_heap_snapshot` | Capture a heap snapshot |
| `wavexis_perf_coverage` | Get JS coverage |
| `wavexis_perf_css_coverage` | Get CSS coverage |

## CSS

| Tool | Description |
| --- | --- |
| `wavexis_css_get_styles` | Get inline and matched styles for an element |
| `wavexis_css_get_stylesheets` | List all stylesheets |
| `wavexis_css_get_rules` | Get CSS rules for a stylesheet |
| `wavexis_css_get_computed` | Get computed styles for an element |

## Debugging

| Tool | Description |
| --- | --- |
| `wavexis_debug_set_breakpoint` | Set a breakpoint by source location |
| `wavexis_debug_set_breakpoint_function` | Set a breakpoint by function name |
| `wavexis_debug_remove_breakpoint` | Remove a breakpoint |
| `wavexis_debug_step_over` | Step over |
| `wavexis_debug_step_into` | Step into |
| `wavexis_debug_step_out` | Step out |
| `wavexis_debug_pause` | Pause execution |
| `wavexis_debug_resume` | Resume execution |
| `wavexis_debug_get_listeners` | Get event listeners on an element |
