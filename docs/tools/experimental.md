# Experimental Tools (20)

Enable with `--caps=experimental`.

## Service Workers

| Tool | Description |
| --- | --- |
| `wavexis_sw_list` | List registered service workers |
| `wavexis_sw_unregister` | Unregister a service worker |
| `wavexis_sw_update` | Trigger service worker update |

## Animations

| Tool | Description |
| --- | --- |
| `wavexis_animation_list` | List running animations |
| `wavexis_animation_pause` | Pause all animations |
| `wavexis_animation_play` | Resume all animations |
| `wavexis_animation_seek` | Seek all animations to a specific time |
