# Emulation Tools (9)

Enable with `--caps=emulation`.

| Tool | Description |
| --- | --- |
| `wavexis_emulate_device` | Emulate a device (iPhone, Pixel, etc.) |
| `wavexis_set_viewport` | Set viewport size |
| `wavexis_set_geolocation` | Set geolocation override |
| `wavexis_set_timezone` | Set timezone |
| `wavexis_set_dark_mode` | Enable/disable dark mode |
| `wavexis_set_locale` | Set locale |
| `wavexis_set_cpu_throttling` | Throttle CPU |
| `wavexis_set_touch` | Enable/disable touch emulation |
| `wavexis_set_sensors` | Set sensor values (accelerometer, gyroscope) |
