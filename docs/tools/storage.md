# Storage Tools (13)

Enable with `--caps=storage`.

| Tool | Description |
| --- | --- |
| `wavexis_localstorage_get` | Get localStorage item |
| `wavexis_localstorage_set` | Set localStorage item |
| `wavexis_localstorage_delete` | Delete localStorage item |
| `wavexis_localstorage_clear` | Clear all localStorage |
| `wavexis_localstorage_list` | List all localStorage keys |
| `wavexis_sessionstorage_get` | Get sessionStorage item |
| `wavexis_sessionstorage_set` | Set sessionStorage item |
| `wavexis_sessionstorage_delete` | Delete sessionStorage item |
| `wavexis_sessionstorage_clear` | Clear all sessionStorage |
| `wavexis_sessionstorage_list` | List all sessionStorage keys |
| `wavexis_cache_storage_list` | List Cache Storage entries |
| `wavexis_cache_storage_delete` | Delete Cache Storage entry |
| `wavexis_storage_state_save` | Save storage state (cookies + storage) |
| `wavexis_storage_state_restore` | Restore storage state |
